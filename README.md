# Cashiely

**Cashiely** is a modern, student-focused finance management web application designed to help college students track expenses, set budgets, and gain insights into their spending behavior. Cashiely emphasizes simplicity, clarity, and actionable insights, making personal finance management easy and approachable for young adults.

---

## Features

- **Smart Expense Logging**: Quickly add, edit, and delete expenses with categories like Food, Transport, Subscriptions, and Academic Supplies.
- **Budget Management**: Set monthly budgets and track your remaining balance and daily spending allowance.
- **Insights & Analytics**: Visualize spending patterns with interactive charts and receive insights to improve financial habits.
- **Cashiely Score**: A gamified metric that tracks your financial discipline.
- **Student-Centric Design**: Minimalist UI with student-friendly categories and simple navigation.

---

## Tech Stack

- **Frontend**: HTML5, CSS3, JavaScript
- **Backend**: Python, FastAPI
- **Database**: MySQL
